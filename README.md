# Homebrew Tap

Official Homebrew tap for pivotings tools.

## Installation

```bash
brew tap pivotings/tap
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `ecto` | Next.js reconnaissance tool |
| `glint` | GitHub OSINT tool for analyzing commit history and user activity |

## Installing Formulas

```bash
# Install individual tools
brew install pivotings/tap/ecto
brew install pivotings/tap/glint

# Or after tapping
brew tap pivotings/tap
brew install ecto glint
```

## Upgrading

```bash
brew update
brew upgrade ecto glint
```

## Uninstall

```bash
brew uninstall ecto glint
brew untap pivotings/tap
```
